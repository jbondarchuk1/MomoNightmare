Bat Swarm Particles is a collection of shader animated bat particle effects. Easily drag and drop into the scene. Uses the standard Unity particle system, so that it is easy to adjust or change the effects.

Features
Includes ShaderGraph shaders
Customizable
LowPoly
Demo scene to easily browse the effects
Medium resolution textures suited for particles


Notes.
When scaling particles the material flap properties will also need to be adjusted accordingly.


Uses ShaderGraph shaders, should be easy to retarget to URP or HDRP.
1. Change to URP or HDRP by double clicking the shader file and opening the graph editor.
2. In the Graph Inspector Settings
3. Add or change active targets to Universal then click Save Asset.
